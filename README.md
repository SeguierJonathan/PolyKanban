PolyKanban est un projet de démonstration technique consistant à développer une API de gestion de tableau Kanban avec plusieurs implémentations backend.

L’objectif est de concevoir un contrat API unique puis de le décliner dans différents langages tels que JavaScript (Node.js), Go, Java et .NET, tout en conservant une cohérence stricte entre les implémentations.

Chaque backend respecte :

- les mêmes routes
- les mêmes formats de données
- la même logique métier

Le projet permet de mettre en avant :

- la conception d’API REST
- la structuration d’une architecture (controllers, services, repositories)
- la maîtrise de plusieurs environnements backend
- la capacité à standardiser et reproduire une solution technique
